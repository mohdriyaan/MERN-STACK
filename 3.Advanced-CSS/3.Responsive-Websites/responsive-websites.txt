These days there are many different screen sizes:- Desktops,Laptops,Ipads,Mobile Phones,etc., and we want our websites to look good on all of them.
And in order to do that we make our websites responsive, so it responds to the changes in the screen sizes.

There are 4 ways to do this:-
1. Media Queries
2. CSS Grid
3. CSS Flexbox
4. External Frameworks e.g. Bootstrap
